# 📬 MailTerminal: Encrypted CLI Client

**MailTerminal** — это легковесный консольный почтовый клиент на Java, специально разработанный для работы с зашифрованными письмами через протокол IMAP.

---

## 🌟 Основные возможности

- 🚀 **Быстрый доступ:** Мгновенное получение последних писем из папки INBOX (Mail.ru и др.).
- 🔐 **Криптография AES:** Встроенная поддержка расшифровки сообщений «на лету» с использованием 16-значных ключей.
- 📂 **Фильтрация:** Специальный режим просмотра только зашифрованных писем (по метке `[ENCRYPTED]`).
- 🎨 **ASCII-интерфейс:** Олдскульный терминальный дизайн с цветовой индикацией статусов.
- 📧 **Отправка и ответы:** Возможность быстро ответить собеседнику, не выходя из терминала.

## 🛠 Технологии

- **Язык:** Java
- **Библиотеки:** 
  - `Jakarta API` (для работы с IMAP/SMTP)
  - `JCE (Java Cryptography Extension)` (для AES шифрования)
- **Протоколы:** IMAPS (порт 993), SSL.

## 🚀 Быстрый старт

### 1. Подготовка конфигурации
Для работы приложения создайте файл `config.txt` в корневой папке проекта со следующей структурой:

```text
Логин:
ваш_email@mail.ru
Пароль:
ваш_пароль_приложения
Ключ шифрования:
16_значный_ключ_AES
```

> **Важно:** Для Mail.ru и Gmail используйте "Пароль для внешних приложений" вместо основного пароля от аккаунта.

### 2. Запуск
Скомпилируйте и запустите проект:
```bash
javac MailTerminal.java
java MailTerminal
```

## 🛡 Безопасность
Приложение использует **AES/ECB/PKCS5Padding** для дешифровки содержимого. Если стандартный ключ из `config.txt` не подходит, программа предложит ввести альтернативный ключ вручную в процессе чтения сообщения.
